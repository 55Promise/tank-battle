# 🎮 坦克大战 (Tank Battle)

经典 FC 坦克大战网页版，使用纯 HTML5 Canvas + JavaScript 实现。

## 🎯 游戏玩法

- **目标**：消灭所有敌方坦克，保护你的基地（🦅）不被摧毁
- **操作**：方向键移动，空格键射击，P 键暂停
- **关卡**：每消灭所有敌人进入下一关，难度递增
- **生命**：3 条命，基地被毁或生命耗尽则游戏结束

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| `↑ ↓ ← →` / `W A S D` | 移动坦克 |
| `空格` | 发射子弹 |
| `P` | 暂停 / 继续 |

## 🧱 地图元素

- 🧱 **砖墙**：可被子弹摧毁
- 🛡️ **钢铁**：不可摧毁
- 🌊 **水域**：坦克不能通过（敌人不可过水，玩家可选）
- 🦅 **基地**：保护目标，被摧毁即失败

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可游玩！

```bash
# 或使用任意本地服务器
npx serve .
python -m http.server 8000
```
